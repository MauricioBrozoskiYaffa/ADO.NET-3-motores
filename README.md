Desarrollo de una aplicación utilizando ADO.NET puro para la conexión y gestión de datos en tres motores de bases de datos: PostgreSQL, MySQL y Microsoft SQL Server. Para el entorno de trabajo se emplean Docker para la ejecución y administración de los servidores de bases de datos, DBeaver para la gestión y consulta de información, y Visual Studio Code como entorno de desarrollo. El proyecto permite realizar operaciones CRUD y validar la conectividad entre la aplicación y múltiples motores de bases de datos mediante una misma tecnología de acceso a datos.
